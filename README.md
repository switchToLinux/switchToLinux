## 学习Linux系统

Linux系统是一个开源的操作系统，它的特点是：
* 免费：Linux系统是免费的，你可以免费使用它。
* 开源：Linux系统是开源的，。
* 安全：Linux系统是安全的，代码透明，这意味着邪恶的人找漏洞与正义的人修漏洞的机会都平等，而大多数人往往会站在正义的一方，每一次发现的漏洞都会被快速修复，并且让漏洞越来越少。
* 稳定：Linux系统是稳定的，你可以保证你的系统稳定。
